# Readium OPDS

A parser for OPDS 1.x and 2.0 written in Kotlin using the [Readium Web Publication Manifest](https://github.com/readium/webpub-manifest).

## Features

- [x] Abstract model
- [x] OPDS 1.x support
- [x] OPDS 2.0 support
- [ ] Search
- [x] Full entries
- [x] Facets
- [x] Groups
- [x] Indirect acquisition
- [ ] Library specific extensions
